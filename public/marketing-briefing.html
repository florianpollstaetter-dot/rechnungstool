3 P0-Bugs blockieren den Start

Diese drei Items werden gerade von Engineer in einem PR gefixt — geschätzte Gesamt-ETA: ~1 Stunde Arbeit, dann Vercel-Deploy.

1 · robots.txt + sitemap.xml broken

Beide URLs antworten HTTP 307 → /login. Google + Bing können nicht crawlen. Indexierungs-Killer.

Fix: Next.js-Middleware-Whitelist + app/robots.ts + app/sitemap.ts.

2 · AGB §1.2 widerspricht Trial-Funnel

Aktuelle AGB-Klausel schließt öffentliche Registrierung aus. Self-Service-Trial juristisch nicht abgedeckt.

Fix: Klausel-Patch von Law Agent vorgegeben, 1:1 deployen.

3 · Datenschutzerklärung Zeile 142

Sagt explizit „kein Tracking, keine Werbung". Damit darf kein einziger Conversion-Pixel oder Cookie-Banner laufen.

Fix: Tracking-Hinweis + Cookie-Consent-Verweis von Law Agent vorgegeben.

Ohne diese drei Fixes ist weder organischer Traffic noch jede Form von Paid-Marketing möglich. Engineer-Issue: SCH-910 (high-priority, ~1h).

Compliance-Findings vom Law Agent

⚠️ niedrig-mittel Markenrecht „Orange Octo"

„Orange Octo" als Wortmarke öffentlich nirgends auffindbar — aber „Octo"-Stamm im SaaS-Space stark belegt (Octobat = Konkurrent ⚠️, Octopus Deploy, OctoAI). Domain orange-octo.com ist unsere; .de/.at/.eu + orangeocto.de/.eu alle frei.

Empfehlung: defensive Domain-Registrierung sofort (~€55/J), Marken-Recherche-Anwalt (€250–600) vor signifikantem Werbespend, optional EU-Wortmarke (~€2.000).

❌ illegal Steuerberater-Affiliate Phase 2

Geplantes Rev-Share-Programm für Steuerberater verletzt §9 BOStB + §57a StBerG (StB dürfen keine Provisionen für Mandantenvermittlung annehmen). OLG Köln: Online-Vermittlungsprovision ist wettbewerbswidrig — wir würden uns selbst dem UWG §3a aussetzen.

Re-Design: Reseller-Modell, Co-Marketing, oder StB-Endkunden-Rabatt. Kein Phase-0/1-Blocker, aber Phase 2 darf nicht starten ohne Re-Design.

Cold-E-Mail DACH

Personalisierte Adressen brauchen Double-Opt-in. Funktionspostfächer (info@, kontakt@) + sachlicher Bezug + Opt-out reichen für §7 UWG. AT (TKG §174) noch strikter — Einwilligung fast immer Pflicht.

LinkedIn-Outreach

Strikt manuell, keine Automation-Tools (Phantombuster, Dux-Soup, Expandi) — verletzt LinkedIn-AGB §8.2 → Account-Sperre. Beeinflusst die geplante Sales-Agent-Automation in Phase 2.
